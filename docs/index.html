SleepWithAI

SleepWIthAI(A.K.A. SWAI) is application for users who want to sleep efficient with personal optimized data.

SWAI will recommend and turn on the alarm at best time to wake up.

Now available on Github.

  • Get Started

Normal Sleep Mode

Normal Sleep Mode is for who needs to wake up at certain time.
User can set time at Home tab with date picker.
After clicking 'Sleep Now' button, SWAI will automatically record your sleeping sound and classify with CoreML machine learning model.

I Don't Care Mode (A.K.A. IDC)

I Don't Care Mode is for who doesn't need to wake up at certain time.
Very opposite from Normal Sleep Mode, user doesn't need to set time for it.
SWAI will automatically get optimized sleep duration time and wake user up.

Set Your Age

You can set your age information for IDC mode.
Only with your age data, SWAI can recommend you best time to wake up, based on 'Sleep Foundation's article'

Change Alarm Sound

You can change your alarm sound on Settings.
Chillin' beat will wake you up softly.

Sleeping Sound Data Analysis

If you press 'Sleep Now' button, SWAI will start recording your sleeptalking & snoring sounds.
With CoreML sound classificatoin machine learning model, SWAI will give you intuitive sleep quality indicators by daily & total chart view.
